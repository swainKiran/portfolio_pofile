Project: Small Pizza Shopping Website

Overview

The Small Pizza Shopping Website is a web application I developed to facilitate online ordering for a local pizza shop. This project showcases my skills in Python, Django, SQL, HTML, and CSS, and includes essential features such as user authentication, menu display, shopping cart functionality, order management, and an admin panel for managing pizza items and orders.

Objectives

1. User Authentication: Implement secure user registration and login functionalities.
2. Menu Display: Present a list of available pizzas with detailed information.
3. Shopping Cart: Enable users to add pizzas to a cart, view the cart, and proceed to checkout.
4. Order Management: Handle the entire order lifecycle, from placement to status tracking.
5. Admin Panel: Provide an interface for administrators to manage pizza items and user orders.

Technologies Used

Python: Core programming language for server-side logic. Django: Framework for building the backend and managing database operations. SQL: Used for database management to store user, menu, and order information. HTML: Provides the structure for the web pages. CSS: Used for styling the web pages to enhance visual appeal and ensure responsiveness.

Key Features

User Authentication:

1. Secure user registration and login with Django's authentication system.
2. User profile management for updating personal information.

Menu Display:

1. Display pizzas with details such as name, description, price, and images.
2. Filter pizzas by categories (e.g., Vegetarian, Non-Vegetarian).

Shopping Cart:

1. Users can add pizzas to their cart, update quantities, and remove items.

2. View cart contents and proceed to checkout for order placement.

Order Management:

1. Users can place orders and view their order history.
2. Track order status from "Pending" to "Delivered".

Admin Panel:

1. Admins can add, update, and delete pizza items.
2. Manage and update the status of user orders.
